GSAn is a new alternative to gene set analysis exploiting the semantic of concepts in the Gene Ontology (GO). Combining with data mining and combinatorial algorithm, GSAn provides a reduced and synthetic number of GO terms resuming an input gene set. In the 1st version, GSAn includes the three ontologies of GO, biological process (BP), molecular function (MF) and cellular component (CC). The sections of interest are:
